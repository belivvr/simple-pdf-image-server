# PDF를 이미지로 바꿔주는 서버

## Methods

### GET

- `.pdf`, `.jpg`, `.jpeg`, `.png` 파일이면 blob을 배열로 내보냅니다.
  - PDF의 경우 페이지가 있기 때문에 페이지 수 만큼 나가지만, 일반 이미지 파일은 배열에 아이템이 하나입니다.
- 그 외에는 경로의 파일/폴더 리스트를 내보냅니다.

### POST

- path에는 경로만 보내고, 파일 이름은 보내온 파일 이름으로 만듭니다.

### PUT

- path로 보내온 경로를 생성합니다.

### DELETE

- 보내온 경로를 삭제합니다.
